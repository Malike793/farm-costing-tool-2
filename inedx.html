خطوات إنشاء موقع مجاني

إنشاء حساب GitHub

1. اذهب إلى github.com

2. اختر "Sign up" وأدخل معلوماتك

3. تحقق من بريدك الإلكتروني

إنشاء مستودع جديد

1. بعد تسجيل الدخول، انقر على "+" في الأعلى

2. اختر "New repository"

3. اسم المستودع: yourusername.github.io

4. اختر "Public" ثم "Create repository"

رفع ملفات الأداة

1. في صفحة المستودع، انقر على "Add file"

2. اختر "Upload files"

3. اسحب وأسقط ملفات HTML, CSS, JS

4. يجب أن يكون الملف الرئيسي باسم index.html

نشر الأداة على GitHub Pages

GitHub Pages تتيح لك استضافة موقع ويب مجاناً مع نطاق فرعي خاص (yourusername.github.io)

تفعيل GitHub Pages

1. اذهب إلى إعدادات المستودع (Settings)

2. اختر "Pages" من القائمة الجانبية

3. في "Branch" اختر main أو master

4. انقر "Save"

الوصول للموقع

1. بعد دقائق قليلة، سيكون موقعك جاهزاً

2. عنوان موقعك: yourusername.github.io

3. يمكنك مشاركة هذا الرابط مع أي شخص

التحديثات

1. عند تعديل ملفاتك، ارفعها مجدداً للمستودع

2. التغييرات تظهر تلقائياً خلال دقائق

3. لا حاجة لإعادة النشر
